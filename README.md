8bitWallet
====
This is local wallet app for [bc4py](https://github.com/namuyan/b4py).

Project setup
----
```
// Project setup
npm install

// Compiles and hot-reloads for development
npm run serve

// Compiles and minifies for production
npm run build

// Lints and fixes files
npm run lint
```
